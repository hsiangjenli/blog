# 📌 介紹
這篇文章說明如何在 Visual Studio Code 使用 Markmap 延伸套件，將 Markdown 語法轉換成心智圖，以便有效地組織與視覺化想法與資訊。

# 🚀 快速開始
## **在 VS Code 安裝此擴充套件**
![](https://i.imgur.com/6YFOq4e.png)
```markmap
# **我的學歷**

## 高職
### KFHS
### 應用外語

## 學士
### NKUST
### 貨幣與銀行學系

## 碩士

#### 資料科學

#### 人工智慧技術與商業應用
#### 機器學習
#### 企業永續管理
#### 生產管理
```

- 使用 Markmap 擴充套件，將 Markdown 文件視覺化為互動式心智圖。
