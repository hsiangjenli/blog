第二版產出的簡報

🔁 重點回顧

  1. 使用 npx @open-slide/cli init my-demo-slide 產生一份完整的簡報專案
  2. 使用 npm run dev 啟動簡報
  3. 使用 /create-slide 產生簡報
  4. 使用 /apply-comments 依據簡報上的 Comment 進行修改
  5. 目前 open-slide 仍在開發中，僅支援匯出成 pdf、html 以及圖片
